# Architecture Decision Records

> 技術選定・設計判断の「なぜ」を記録。Progressive Context として機能する。

## Format: docs/adr/ADR-XXX-{title}.md
```
# ADR-XXX: {タイトル}
## Status: accepted | deprecated | superseded by ADR-YYY
## Date: {YYYY-MM-DD}
## Context: {なぜこの判断が必要だったか}
## Decision: {何を決めたか}
## Alternatives: {検討した他の選択肢}
## Consequences: {結果。良い点と悪い点}
```
